# 📑 Abas Verticais Interativas

Projeto desenvolvido com HTML e CSS criando uma interface com navegação lateral (abas verticais), onde o conteúdo é alterado dinamicamente ao selecionar cada opção.

---

## 🛠️ Tecnologias utilizadas
- HTML
- CSS

---

## 🎯 Objetivo
Praticar a criação de interfaces interativas utilizando apenas HTML e CSS, explorando navegação, organização de conteúdo e estilização.

## 📚 O que foi praticado
- Estruturação de layout com HTML  
- Estilização com CSS  
- Criação de navegação lateral  
- Troca de conteúdo sem uso de JavaScript  
- Uso de inputs (radio) para controle de estado
